# Rich Console Export Refactoring Task

This task requires refactoring the Rich library's Console class to extract export functionality.

## Getting Started

1. Read SPEC.md for requirements
2. Study rich/console.py to understand the export methods
3. Create rich/console_export.py with extracted functionality
4. Update Console methods to delegate to the new module
5. Run tests to verify: `pytest tests/test_console.py -v`
